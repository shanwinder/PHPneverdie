# คู่มือผู้เรียนเบื้องต้น

## เริ่มเรียน

1. เปิดหน้าเว็บ
2. สมัครสมาชิกที่ `/register`
3. ระบบจะพาเข้าสู่ Dashboard
4. กด `เรียนต่อ` เพื่อเริ่มบทเรียนแรก

## Dashboard

Dashboard แสดงข้อมูลส่วนตัวของผู้เรียน:

- Level ปัจจุบัน
- XP
- Progress รวม
- จำนวนบทเรียนที่เรียนจบ
- จำนวน Quiz ที่ผ่าน
- บทเรียนถัดไปที่ควรเรียน

## เรียนบทเรียน

ในหน้า lesson จะเห็น:

- ชื่อบทเรียนและสรุป
- เนื้อหาบทเรียน
- ตัวอย่างโค้ด
- ปุ่มกลับ Learning Path
- ปุ่มทำ Quiz

เมื่อเปิดบทเรียนครั้งแรก ระบบจะบันทึกสถานะ `in_progress`

## ทำ Quiz

1. อ่านบทเรียนให้จบ
2. กด `ทำ Quiz`
3. เลือกคำตอบทุกข้อ
4. กดส่งคำตอบ

ถ้าคะแนนถึงเกณฑ์ผ่าน ระบบจะบันทึกบทเรียนนั้นเป็น `completed` และเพิ่ม XP เฉพาะครั้งแรกที่ผ่าน
